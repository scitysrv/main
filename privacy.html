Политика конфиденциальности

о заключении договора об оказании услуг

Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты персональных данных физических лиц (далее — «Пользователи»), пользующихся услугами Самозанятого Иконникова Дениса Дмитриевича (далее — «Исполнитель»).

1. Общие положения

# Описание
1.1 Исполнитель обрабатывает персональные данные Пользователей в соответствии с Федеральным законом РФ № 152-ФЗ «О персональных данных» от 27.07.2006.
1.2 Настоящая Политика действует в отношении всей информации, которую Исполнитель получает от Пользователей в связи с оказанием услуг.
1.2 Использование услуг означает согласие Пользователя с условиями данной Политики.

2. Обрабатываемые данные

Исполнитель не собирает и не хранит IP-адреса, адреса электронной почты, номера телефонов и иные данные, если только они не предоставлены Пользователем добровольно. Исполнитель может обрабатывать следующие данные:

  • Игровой никнейм Пользователя;
  • Идентификатор Telegram-аккаунта (при авторизации через Telegram-бота);
  • Иная информация, добровольно предоставленная Пользователем при обращении к Исполнителю.

3. Цели обработки данных

Персональные данные обрабатываются исключительно для:

  • Оказания услуг (предоставления игровых статусов на серверах проекта Shield City);
  • Идентификации Пользователя для доступа к услугам;
  • Исполнения обязательств по возврату средств;
  • Ведения налогового учета.

4. Правовые основания обработки

Основанием для обработки персональных данных является заключение договора (публичной оферты) между Пользователем и Исполнителем, а также действующее законодательство РФ.

5. Передача данных третьим лицам

Исполнитель не передает персональные данные третьим лицам, за исключением случаев, прямо предусмотренных законодательством РФ (например, по запросу государственных органов).

6. Хранение и защита данных

# Описание
6.1 Персональные данные хранятся в минимально необходимом объеме и только столько времени, сколько требуется для достижения целей обработки.
6.2 Исполнитель принимает технические и организационные меры для защиты данных от несанкционированного доступа.

7. Права Пользователя

Пользователь вправе:

  • Получать информацию о своих персональных данных, обрабатываемых Исполнителем;
  • Требовать уточнения, блокировки или уничтожения своих персональных данных;
  • Отозвать согласие на обработку данных, направив обращение Исполнителю.

8. Контакты

Для вопросов, связанных с обработкой персональных данных, Пользователь может обратиться к Исполнителю через:

  • Контакт основной (E-mail): scitysrv@yandex.ru;
  • Контакт дополнительный: Официальный Telegram-канал проекта Shield City (с возможностью отправки сообщений владельцу);
